# Progetto_singolo

Progetto singolo Nato Daniele con seguente traccia:

Realizzare un firmware che, a seconda della durata della pressione sul tasto USER, metta il LED in uno dei tre modalità operative:
Blink veloce (≈ 0,5 s ON/OFF) – attivato con press < 1 s.
Lampeggio lento (≈ 2 s ON/OFF) – attivato con press ≥ 1 s e < 2,5 s.
Duty‑cycle 50 % (ON per 500 ms, OFF per 500 ms) – attivato con press > 2,5 s.
